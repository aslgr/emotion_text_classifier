# Emotion Text Classifier

A complete Natural Language Processing (NLP) pipeline for emotion classification using classical machine learning techniques.

This project builds, evaluates, and compares models capable of classifying text into different emotional categories such as *joy, sadness, anger, fear, love,* and *surprise*.

---

## Motivation

Understanding emotions in text is a fundamental problem in NLP with applications in:

- Sentiment analysis
- Social media monitoring
- Customer feedback analysis
- Human-computer interaction

The goal of this project is not only to build a working classifier, but to design a **clean, reproducible, and well-structured machine learning pipeline**, following best practices used in real-world scenarios.
